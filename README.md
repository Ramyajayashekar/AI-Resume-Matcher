# AI Resume Matcher

An AI-powered Resume Screening and Candidate Ranking System built using Python and Streamlit.

## Features

* Resume vs Job Description Matching
* PDF and DOCX Resume Parsing
* Technical Skill Extraction
* Match Percentage Calculation
* Missing Skill Analysis
* Resume Ranking System
* Multi Resume Comparison
* Authentication System
* Interactive Streamlit UI

## Tech Stack

* Python
* Streamlit
* Pandas
* NLP-based Skill Matching
* PyPDF2
* pdfplumber
* python-docx

## How to Run

### Install dependencies

```bash id="xjlwmq"
pip install -r requirements.txt
```

### Run application

```bash id="hh6v1u"
streamlit run app.py
```

## Project Workflow

1. Upload Resume
2. Paste Job Description
3. Extract Skills
4. Compare Resume Skills with JD Skills
5. Generate Match Score
6. Display Missing Skills & Suggestions
